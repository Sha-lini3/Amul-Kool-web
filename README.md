## About

Amul Kool is a creative frontend project that transforms a traditional product landing page into an engaging digital experience.

Instead of a static website, the project combines scroll-driven animation, dynamic flavor themes, glassmorphism, and an interactive checkout interface to create a modern product presentation.

---

## Features

- 🎬 Scroll-driven frame animation
- 🥛 Interactive flavor selection
- 🎨 Dynamic theme switching
- ✨ Glassmorphism interface
- 🚀 Animated preloader
- 🛒 Interactive checkout experience
- 💳 Card & UPI payment interface (UI Demo)
- 📱 Responsive design

## Getting Started

Clone the repository

```bash
git clone https://github.com/Sha-lini3/Amul-Kool-web.git
```

Move into the project

```bash
cd Amul-Kool-web
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
